# Academia Caucana de Taekwondo ITF

Landing page institucional desarrollada con **Vite + React + TypeScript** para la Academia Caucana de Taekwondo ITF.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run type-check
npm run preview
```

## Estructura principal

```text
src/
  assets/
    illustrations/
  components/
    layout/
    sections/
    ui/
  data/
  hooks/
  lib/
  types/
  App.tsx
  main.tsx
  index.css
```

## Notas

- Diseño responsive mobile-first con identidad ceremonial inspirada en diploma.
- Tipado fuerte para navegación, programas, valores, cinturones y datos de contacto.
- Contenido desacoplado en `src/data/site-content.ts` para facilitar iteración futura.
